# CLAUDE.md — Jobish

Карьерный коуч-консультант. Веб-платформа карьерного развития (студенческий проект НИУ ВШЭ, «ИТ-консалтинг»).

## Что это
IT-платформа для двух сегментов: (1) молодые специалисты/выпускники, (2) опытные специалисты, меняющие сферу. Ядро ценности — прозрачный персонализированный путь развития: AI-коуч, roadmap по треку, тесты, аналитика рынка, мотивация.

Первоисточник требований: `полный_рабочий_документ.docx (3).pdf` (118 стр.). Извлечённый текст кэшируется в `/tmp/proj.txt` (можно перегенерировать через pypdf).

## Документация (читать это, а не PDF)
- `docs/project-overview.md` — описание проекта, аудитория, монетизация, бизнес-цели.
- `docs/functionality.md` — функциональность MVP (ФТ 1–7), роли, user stories, критерии приёмки.
- `docs/architecture.md` — стек, архитектура (заявленная в документе vs реализуемая), модель данных, риски.
- `docs/plan.md` — пошаговый план разработки, статус фич, чеклист.
- `docs/audit.md` — аудит требований (нестыковки, риски, открытые вопросы).
- `docs/gap-analysis.md` — сверка «реализовано vs документ» + фазовый план дозакрытия пробелов.

## Текущий статус
**MVP-срез реализован и протестирован** (этапы 1–4 завершены). Стек: Next.js 16 +
Prisma 7 (SQLite) + Tailwind v4. AI-коуч на OpenRouter с мок-фолбэком.
Тесты: 30 unit (Vitest) + 7 E2E (Playwright) — зелёные. Запуск: см. README.md.
Remote: github.com/PieceOfBulka/jobish.
