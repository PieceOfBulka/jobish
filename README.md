# Jobish — карьерный коуч-консультант

Веб-платформа карьерного развития: AI-коуч, профориентация, карта развития
(roadmap), контрольные тесты, аналитика рынка труда и мотивация.
Студенческий проект НИУ ВШЭ («ИТ-консалтинг»). Реализован **вертикальный срез
клиентского MVP** — см. `docs/`.

## Стек
- **Next.js 16** (App Router, React 19, TypeScript)
- **Tailwind CSS v4** — дизайн-система и UI-kit
- **Prisma 7** + SQLite (адаптер better-sqlite3)
- **Аутентификация** — JWT (jose) + bcrypt, httpOnly-cookie
- **AI-коуч** — OpenRouter (с фолбэком на детерминированный мок-движок)
- **Тесты** — Vitest (unit) + Playwright (E2E)

## Быстрый старт
```bash
npm install            # ставит зависимости и генерирует Prisma Client
cp .env.example .env   # при необходимости отредактируйте
npm run db:push        # создаёт SQLite-базу по схеме
npm run db:seed        # наполняет демо-данными (6 профессий + demo-пользователь)
npm run dev            # http://localhost:3000
```

Демо-доступ: **demo@jobish.ru** / **Demo1234!**

## Переменные окружения (`.env`)
| Переменная | Назначение |
|---|---|
| `DATABASE_URL` | строка подключения SQLite (`file:./dev.db`) |
| `JWT_SECRET` | секрет для подписи сессий |
| `OPENROUTER_API_KEY` | ключ OpenRouter; пусто → мок-движок коуча |
| `OPENROUTER_MODEL` | модель (по умолчанию недорогая `meta-llama/llama-3.1-8b-instruct`) |

## Скрипты
| Команда | Действие |
|---|---|
| `npm run dev` / `build` / `start` | разработка / сборка / прод |
| `npm run db:push` / `db:seed` / `db:reset` | работа с БД |
| `npm test` | unit-тесты (Vitest) |
| `npm run test:e2e` | E2E-тесты (Playwright) |
| `npm run lint` | ESLint |

## Реализованная функциональность (MVP)
- Регистрация/вход/профиль, загрузка резюме (мок)
- Профориентационный тест → ранжированные профессии (US4, US5)
- Каталог профессий с аналитикой рынка: зарплаты по грейдам, график, топ-компании (US6)
- Карта развития (roadmap) по грейдам, трекинг навыков, мотивация (US1–3)
- Контрольные тесты с механикой «заморозки» и мок-разморозкой (US7, ФТ-4)
- AI-коуч в чате с лимитом бесплатного тарифа (ФТ-1,2,6)
- Карьерные цели (US9), тарифы с мок-оплатой

## Мокировано (демо-режим, с пометкой в UI)
Платежи/эквайринг, live-данные рынка (сид вместо парсинга hh.ru), email/Zoom,
S3, прод-SSO. Кабинет коуча и полная админка — вне MVP (Release X).
