# Дельта требований v2 (Отчёт команда 6)

Источник: `ИТ-консалтинг (Отчет команда 6)-2.docx`. Текст: `/tmp/req2.txt`.

## Изменения vs v1
- Бизнес-цель: **10 тыс. регистраций** в 1-й год (было 40k), конверсия 20% (>2000 платных).
- **ФТ-3.1**: данные рынка — через **официальный API hh.ru** (раньше скрапинг Scrapy/Selenium).
- **ФТ-1.5**: роль admin назначается вручную через БД/другим админом (раньше универсальный пароль).
- **ФТ-1.2**: профиль расширен: роль (специалист/коуч), образование (вуз+специальность), **текущая должность**, опыт **в месяцах**, навыки-теги.
- US5: ранжированный список профессий выводится **в чат**; при повторном тесте — **два списка** (старый и новый).
- US9/US10: отдельный раздел **«Карьерные цели»** с диагностикой и траекториями.
- US7: персонализированное тестирование, привязанное к roadmap/портрету.

## Этапы (Release X — вне MVP)
US11, US12, US16, US18, US23–US31 (кабинет коуча целиком). Остальные client/admin US — MVP.

## Логическая модель данных (таблицы из ДОКа — целевая схема)
- **User**: role_id, email, is_verified, is_blocked, password_hash, password_algo (bcrypt|argon2|pbkdf2), password_updated_at, created_at, updated_at.
- **Auth_Provider**: code (google|vk|telegram), name, is_active.
- **User_Profile**: full_name (только буквы), telegram_nick (с «@»), age (0–150), experience_months (0–1000), education_place, grade_level (enum: Бакалавр/Магистр/Специалитет/Аспирантура/Среднее проф./Школьное/Самообразование/Другое), current_specialty, current_position (Junior/Middle/Senior/Lead/Head/Другое), salary_expectation, resume_file_url.
- **Test / Test_Question** (question_type: single_choice|multiple_choice|scale|text, weight 0..1, сумма=1) / **Test_Attempt**.

## ФТ — критерии приёмки (ключевое)
- ФТ-1.1: email (валидация, уникальность), пароль >8 с цифрами/спецсимволами/буквами, **подтверждение пароля**; SSO (google/vk/telegram).
- ФТ-1.6: верификация почты (код/ссылка).
- ФТ-2.1: профтест 30–60 вопросов, сохранение прогресса, история; админ-наполнение.
- ФТ-2.2: отчёт 3–5 профессий (макс 5), % соответствия, **обоснование**, порог ≥70% (иначе пометка «явных совпадений нет»).
- ФТ-2.3: типы вопросов single/multiple/text.
- ФТ-2.4: отчёт теста — %, сильные/слабые темы, **рекомендации курсов**, вывод.
- ФТ-3.1: 7 разделов сводки (описание, аналитика с медианой/перцентилями+город, сложность перехода, hard/soft, компании, задачи/кейсы).
- ФТ-3.2: поиск вакансий (профессия/город/компания/формат/вилка/опыт/занятость). ФТ-3.3: персональные вакансии.
- ФТ-4.1–4.3: текущий/целевой статус, карточки-этапы с источником (название/ссылка/автор/тип/длительность/рейтинг)+сроки; **конструктор** (создать/изменить/удалить/двигать).
- ФТ-6.1: чат — диалог, анализ резюме, обзор профессии, подготовка к интервью, мотивация, трекинг.
- ФТ-7.1–7.5: передача прогресса через чат → обновление roadmap + карьерного портрета + мотивационный блок.
- ФТ-7.7: генерация персональных тестов (1–10 вопросов). ФТ-7.8: при <50% — поддержка + снижение сложности.
- НФТ: SC3 — удаление данных по запросу; SP3 — форма поддержки (ответ ≤24ч); U3 — RU/EN.
