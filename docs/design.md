# Дизайн-система Jobish

Материалы для презентации. Готовые ассеты:
- `public/logo.svg` — логотип (рыбка: **job + fish**) + вордмарк.
- `public/design-system.svg` — обзорный лист (палитра, типографика, кнопки, карточки) для вставки в слайд.

Реализация: Tailwind CSS v4, токены в `src/app/globals.css`. Шрифт — **Inter** (латиница + кириллица).

## 1. Бренд
- **Идея логотипа:** Jobish = *job* + *fish*. Знак — рыбка в индиго-плашке со скруглением 12px, рядом вордмарк «Job**ish**» (суффикс акцентным индиго).
- **Тон:** профессионально, дружелюбно, с заботой о пользователе; чистые поверхности, много воздуха, мягкие тени, плавные анимации.

## 4. Компоненты (UI-kit)
- **Кнопки:** `primary` (индиго, тень-свечение), `accent` (бирюзовый), `outline` (бордер), `ghost`. Радиус 12px, состояния hover/active/focus-ring, disabled.
- **Карточки:** `card` (бел. фон, бордер slate-100, мягкая тень), `card-hover` (подъём + усиленная тень).
- **Поля ввода:** `input` (радиус 12px, focus-ring индиго), `label`.
- **Бейджи:** `badge` (pill) — статусы спроса, тарифа, прогресса.
- **Графики:** Recharts — динамика зарплат по грейдам (линии), прогресс-кольцо на карте.
- **Анимации:** `fade-up` (появление контента), плавные transition 200ms.

## 5. Ключевые экраны (для слайдов)
**Публичные:**
- Лендинг — hero «Прозрачный путь к карьере мечты», блоки сегментов, 7 возможностей (ФТ 1–7), «как это работает», CTA.
- Каталог профессий + детальная страница (аналитика рынка: графики зарплат, топ-компании, навыки по грейдам, сложность перехода, кейсы).
- Тарифы (Старт/Оптимальный/Про + бесплатный).

**Личный кабинет (сайдбар + контент):**
- Дашборд — стрик, прогресс-кольцо, быстрые действия (кликабельные карточки).
- AI-коуч — чат с историей диалогов (список слева), Markdown в ответах.
- Профориентация — пошаговый тест → ранжированные профессии; при повторном входе сразу карта треков.
- Карта развития (roadmap) — этапы (Основы / Уверенный / Продвинутый уровень), статусы навыков, конструктор (добавить/удалить/двигать).
- Вакансии — поиск с фильтрами.
- Тесты — контрольные + персональный тест; автоотметка этапа при сдаче.
- Профиль — данные, навыки-теги, карьерные цели, карьерный портрет.

**Админ-панель:** обзор, пользователи (модерация), тарифы, рассылки.

## 6. UX-принципы
- Ключевые действия — за 3–5 шагов (NFR_U1).
- Адаптивность: desktop / tablet / mobile (NFR_U2); сайдбар сворачивается в бургер на мобильном.
- Доступность: контраст текста, focus-ring, связанные label↔input, aria-метки на иконках-кнопках.
- Демо-режим помечен в UI (платежи, данные рынка, верификация почты).

## 7. Как использовать в презентации
1. Вставьте `public/logo.svg` на титул и в шапку слайдов.
2. Слайд «Дизайн-система» — `public/design-system.svg` целиком.
3. Палитру/типографику возьмите из таблиц выше.
4. Скриншоты экранов снимите с запущенного приложения (`npm run dev`, http://localhost:3000) — список экранов в разделе 5.
