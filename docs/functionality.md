# Функциональность — Jobish

Этапы: **MVP** vs **Release X** (последующие релизы).

## Роли
- **Клиент** — основной пользователь (большинство в MVP).
- **Администратор** — наполнение/модерация, аналитика, тарифы, тикеты (часть MVP, большая часть Release X).
- **Карьерный коуч** — консультации, ведение клиентов (почти весь Release X).

## MVP — функциональные блоки (ФТ 1–7)
1. **Чат-диалог** — текстовый чат с ботом, отвечает с учётом контекста, целей и задач пользователя.
2. **AI-коуч / профориентация** — профориентационные сессии, ответы по навыкам и компетенциям для начинающих и опытных.
3. **Roadmap (карта развития)** — по выбранному треку, с желаемыми и необходимыми компетенциями.
4. **Контролирующие теоретические тесты** — проверка прохождения этапов roadmap; механика «заморозки» на период при неудаче.
5. **Рекомендации материалов** — что изучить для освоения навыка из карты.
6. **Аналитика рынка труда** — дашборд (зарплаты, востребованность, пороги входа, топ-компании).
7. **Трекинг прогресса + мотивация** — удержание фокуса, ежедневная привычка (вход в приложение).

## Release X (вне MVP)
Подбор менторов/коучей и запись на консультации (US12), соцсеть/сообщества, интеграция с агрегаторами вакансий и партнёрами, аналитика успешных кейсов найма, кастомизация roadmap, аватар ассистента, вводные мини-курсы, предсказание рынка/позиции, мок-интервью, синхронизация прогресса с внешними площадками.
