# Аудит требований — Jobish

## 1. Главное: разрыв масштаба
Документ описывает распределённую enterprise-систему (5 чел., 11 мес., 32.5 чел.-мес.). Полный объём в одной сессии нереализуем. Предложение: строить **вертикальный срез клиентского MVP** — реальное работающее приложение по ключевым потокам, монолит вместо микросервисов, моки вместо нереализуемых интеграций. Требует согласования границ.

## 2. Нестыковки бизнес-логики
1. KPI требует ≥10% коучей в 1-й год, но весь функционал коуча = Release X — привлекать нечем.
2. KPI неполны/опечатки («25 млн» без валюты/горизонта, «не стоит строгого KP» = KPI). Опытные специалисты — ключевой сегмент, но без KPI.
3. Сбой нумерации US11–US12 в таблице (строка «12» пустая, контент уехал).
4. Механика «заморозки» тестов + монетизация попыток/рекламой конфликтует с retention 40% / NPS≥35.
5. Целевые метрики агрессивны для нового B2C (помечены как корректируемые — ОК).
6. Тесты генерит ИИ, но на их результате стоит блокировка прогресса → риск несправедливой заморозки при галлюцинации. Нужна модерация банка вопросов.

## 3. Технические риски
1. **152-ФЗ ↔ OpenAI/S3**: НФТ требует и GDPR, и 152-ФЗ (локализация ПДн в РФ), а стек — GPT-4o-mini (OpenAI/США) + S3. Конфликт. Прод: GigaChat/YandexGPT + сервер РФ.
2. **Скрапинг hh.ru** через Scrapy/Selenium с «обходом блокировок» против ToS → официальный API.
3. **Себестоимость LLM 0.029 ₽/запрос (97% маржа)** занижена — не учтён контекст диалога/roadmap/генерации тестов.
4. **99.5% доступности при одной PostgreSQL** — SPOF, нужна HA-реплика + фейловер.
5. **ML-инженер 0 FTE в апреле, 2.9 чел.-мес. всего** — мало для AI-ядра.

## 4. Дизайн/референсы
- Canva — приватная `/edit` ссылка, вероятно недоступна; нужен экспорт/публичная ссылка.
- Figma «Untitled» node 0-1 — похоже на пустой файл. Попробую Figma MCP; иначе интерпретирую по описанию (split-screen, карточки профессий, roadmap, дашборд, UI-kit).

## 5. Предложение по объёму
Вертикальный срез клиентского MVP: auth/профиль, профтест→профессии, roadmap, прогресс/мотивация, теоретический тест с заморозкой, дашборд рынка (сид), AI-чат (мок/LLM). Тарифы — экран с мок-платежами. Тесты: Vitest + Playwright. Вне объёма: Kong, микросервисы, реальный скрапинг, S3, прод-SSO, кабинет коуча, полная админка.
